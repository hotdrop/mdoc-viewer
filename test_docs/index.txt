## 使い始めのチェックリスト

1. `LOCAL_DOCS_ROOT` がこのフォルダを指していることを確認。
2. ブラウザでトップページを開き、見出しやリストのレンダリングを確認。
3. 上記リンクを辿って、相対パス解決と`.txt`強制挙動をテスト。

### サンプルの目的

- Markdownレンダリングの基本スタイル確認
- サニタイズと危険なリンクの除外を検証
- パス正規化・`index.txt`優先の挙動を確認
- ETagや304レスポンスのキャッシュ確認用テキストを提供

```bash
pnpm dev
```
